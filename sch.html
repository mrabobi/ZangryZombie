Zangry Zombie

In prima parte a proiectului am facut desing-ul jocului si am realizat interfata acestuia. Am facut cate o pagina html pentru magazinul jocului, pentru indicatiile utile jucatorului, si pentru top-ul jucatorilor.

Urmatoarea etapa a fost implementarea jocului si a functionalitatilor. Jocul a fost realizat in javascript, iar microserviciile in php. In folderul api avem create mai multe API-uri prin care interactionam cu baza de date.

In fisierul register.php, apelez un API-ul care insereaza in baza de date conturile noi create, daca un username se afla deja in baza de date atunci va afisa un mesaj, acest microserviciu v-a fi apelat in fisierul register1.php unde se afla si form-ul pentru crearea contului. In login.php se apeleaza acelasi API facut pentru tabela users, si se verifica username-ul si parola din baza de date cu cele introduse in pagina de login index1.php unde este apelat acest microserviciu. Pe acest model am creat si celelalte functionalitati pentru shop si pentru top score.

In index.php se afla pagina principala si este leagata toata logica jocului continand un meniu cu butoane care te directioneza spre celelalte servicii si butonul de start care va incepe jocul.

Arhitectura jocului respecta MVC.Clasa game tine toata logica jocului si include mai multe functii de update pentru diferite aspecte de gameplay(pozitia jucatorului,viata jucatorului,pozitia inamicilor etc.) si functia ce aduce jocul in starea initiala.Clasa display include functii pentru a desene pe ecran toate elementele jocului(jucatorul,inamicii,harta).Clasa input inregistreaza tastele apasate de jucator si apeleaza functii corespunzatoare din clasa game sau din clase ce sunt componente ale clasei game. Pe langa acestea 3,mai gasim si clase pentru a modela jucatorul,inamicii si harta.Acestea retin elemente ce tin de entitatile respective,cum ar fi numarul de vieti ale jocatorului,stats-urile caracterului (damage,crit chance etc),layout ul hartii si pozitia fiecarui inamic.
